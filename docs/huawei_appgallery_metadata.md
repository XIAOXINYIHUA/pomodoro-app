## 应用描述（中文，建议 500-800 字）

```
🍅 番茄鸡 — 用番茄钟管理专注，用运动计划保持活力！

忙碌的每一天，既要高效完成工作，也要保持身体健康。番茄鸡将经典番茄工作法与运动健康管理合二为一，帮助你平衡工作与生活。

【核心功能】

🍅 番茄钟专注
• 环形倒计时器，支持专注 / 短休息 / 长休息自动切换
• 自定义时长：专注 25 分钟、休息 5 分钟，或按需调整
• 专注计划：创建多个计时方案（如"深度学习 50分钟"、"快速冲刺 15分钟"），一键切换
• 关联任务：选择当前任务开始专注，每个番茄都被记录
• 后台计时 + 完成通知，锁屏也能准确提醒

📋 待办事项
• 首页记事本风格待办列表，快速记录
• 顶部显示当前日期和星期
• 勾选完成自动添加删除线
• 支持滑动删除、长按编辑

🏃 运动计划
• 创建个人运动计划（跑步、力量、瑜伽、骑行、游泳等）
• 自定义时长、频率、时间段
• 一键打卡，运动日志自动记录
• 连接系统健康应用，自动获取步数、卡路里、心率等数据

📊 统计分析
• 今日专注概览：番茄数、专注时长、完成任务数
• 本周专注趋势折线图（fl_chart）
• 运动完成率统计
• 每日数据自动汇总

⚙️ 更多亮点
• 深色 / 浅色 / 跟随系统主题
• 中英双语切换
• 数据完全本地存储，无需联网，隐私安全
• 数据导出为 CSV 文件
• 纯离线使用，无需注册账号

【适用场景】
• 学生备考：番茄钟管理学习时间，拒绝拖延
• 远程办公：保持专注节奏，按时休息
• 健身计划：制定运动计划，记录每日运动量
• 日常效率：用待办清单规划一天的待办事项

【隐私保护】
番茄鸡是一款完全离线的应用。所有数据均存储在您的设备本地，不会上传到任何服务器。不需要注册账户，不收集个人信息。

【即将推出】
• 华为运动健康数据同步
• 云端数据备份
• 桌面小组件
• 更多自定义主题
```

## 应用描述（英文，建议 300-500 字）

```
🍅 Tomato Chicken — Manage focus with Pomodoro Timer, stay active with Workout Plans!

【Key Features】

🍅 Pomodoro Timer
Circular countdown timer with auto-switching between focus / short break / long break. Customizable durations. Create multiple focus plans for different scenarios.

📋 To-Do List
Notepad-style task list on the home screen. Add, edit, complete, and swipe to delete. Auto-strikethrough for completed items.

🏃 Workout Plans
Create personalized workout plans (Running, Strength, Yoga, Cycling, Swimming, Walking). Log workouts with one tap. Connect to Apple Health / Google Health Connect for automatic step, calorie, and heart rate tracking.

📊 Statistics
Daily focus overview, weekly trend charts, workout completion tracking. All data summarized automatically.

⚙️ More
Dark / Light / System theme. Chinese / English bilingual. Fully offline — no account needed, all data stored locally. CSV export supported.

【Privacy】
100% offline. No data collection. No account required. Your data stays on your device.
```
